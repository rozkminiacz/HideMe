Welcome to HideMe - Android Steg app

Now you can hide plain text in picture you've just taken!

Steganography never been easier!

But first, read this basic tutorial:

How to hide text

  • Open image from gallery...
  • ... or take image!
  • Set text to hide
  • Set canal (RGB)
  • Click "apply" to hide text
  • Save your image!
  • Now you can share photo via email, Skype, etc...

How to reveal message

  • Open image from gallery
  • Choose canal (RGB)
  • Click "decode"
  • Voila!
